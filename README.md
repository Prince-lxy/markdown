### 1 新增 template.md 文件为 tmp.md

将 markdown 目录下 template.md 文件拷贝为 tmp.md 文件，后续对 tmp.md 文件进行修改，避免破坏模板。

```
cp ${somewhere}/markdown/template.md tmp.md
```
